### 1.2 示例：Flex水平垂直居中

```markdown
# Flex水平垂直居中

## 效果
让子元素在父容器中水平和垂直都居中

## 完整代码
```css
.parent {
    display: flex;
    justify-content: center;  /* 水平居中 */
    align-items: center;      /* 垂直居中 */
    width: 300px;
    height: 200px;
    background: #f5f5f5;
}

.child {
    width: 100px;
    height: 50px;

## 适用场景
- 登录框居中
- 加载动画居中
- 弹窗内容居中
- 卡片内图标居中

## 来源

### 3.2 知识点示例：Flexbox布局

**文件**：`knowledge/css/Flexbox布局.md`

---

# Flexbox布局

## 一句话解释
Flexbox是一种一维布局模型，让元素在一条轴线上灵活排列。

## 详细说明
- Flexbox解决了传统布局（float、inline-block）的各种痛点
- 特别适合：居中、等分、自适应布局
- 核心思想：父容器控制子元素的排列方式

## 核心概念
- **容器（flex container）**：设置 `display: flex` 的父元素
- **项目（flex item）**：容器内的直接子元素
- **主轴（main axis）**：项目排列的方向（默认水平）
- **交叉轴（cross axis）**：垂直于主轴的方向

## 代码示例

### 基础用法
```css
.container {
    display: flex;
    justify-content: center;  /* 主轴居中 */
    align-items: center;      /* 交叉轴居中 */
}
```

### 实际项目中的应用
```css
/* ranphicat-website 导航栏 */
.navbar {
    display: flex;
    justify-content: space-between;  /* 两端对齐 */
    align-items: center;             /* 垂直居中 */
}
```

## 实际应用
在ranphicat-website项目中：
1. 导航栏：logo和菜单两端对齐
2. 产品卡片：水平排列，自动换行
3. 页脚内容：水平分布

## 容易混淆的地方
- `justify-content` 控制主轴，`align-items` 控制交叉轴
- 默认主轴是水平的，但设置 `flex-direction: column` 后会变
- `gap` 是子元素间距，不是外边距

## 学习心得
- 刚开始总是搞混 justify-content 和 align-items
- 后来记住：justify 是"证明合理"，主轴是主要的，所以 justify 控制主轴
- 实际写几个例子就明白了，不要死记

## 相关知识点
- [[justify-content]] - 主轴对齐方式
- [[align-items]] - 交叉轴对齐方式
- [[flex-direction]] - 主轴方向
- [[flex-wrap]] - 换行设置

## 参考资料

- 文档：https://developer.mozilla.org/zh-CN/docs/Web/CSS/flex

### 3.3 知识点示例：div和span的区别

**文件**：`knowledge/html/div和span的区别.md`

# div和span的区别

## 一句话解释
div是块级元素，独占一行；span是行内元素，只占内容宽度。

## 详细说明
- div用于大块内容的容器
- span用于小段文字或图标的容器
- div可以包含div和span，span一般不包含div

## 核心概念
- **块级元素**：独占一行，可以设置宽高
- **行内元素**：只占内容宽度，不能设置宽高

## 代码示例
```html
<!-- div：独占一行 -->
<div>这是第一个div</div>
<div>这是第二个div</div>

<!-- span：不换行 -->
这是<span style="color: red;">红色</span>文字。
这是<span style="color: blue;">蓝色</span>文字。
```

## 实际应用
在ranphicat-website项目中：
- 导航栏用 `<nav>`（语义化的div）
- 产品卡片内的标签用 `<span>`

## 容易混淆的地方
- span设置width和height不生效（因为它是行内元素）
- 如果想让span设置宽高，要加 `display: inline-block`

## 学习心得
- 记住：div是盒子，span是标签
- 盒子可以装东西，标签只能贴在文字上

- [[行内元素]]
- [[语义化标签]]

## 参考资料
- 文档：https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element/div

### 模板使用方法

**方法1：手动创建**
1. 在Obsidian中新建文件
2. 复制上面的模板内容
3. 填写实际内容

**方法2：Obsidian模板插件**
1. 设置 → 核心插件 → 模板 → 开启
2. 设置模板文件夹路径
3. 把模板文件放到模板文件夹
4. 快捷键一键插入模板
